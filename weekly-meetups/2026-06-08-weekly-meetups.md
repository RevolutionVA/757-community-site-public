#### Claude + Browser Automation: Your AI Sees Your Screen | AICHR

- **Time:** 07:00 PM
- **Group:** 757 Artificial Intelligence Collective
- **Link:** [Event Link](https://www.meetup.com/aicollectivehr/events/314896365/)

**Description:**
**Notice:** This meeting will be recorded. By joining and remaining in this meeting, you consent to being recorded. If you do not consent, please do not join.

This event is capped at 500 attendees. Register now to secure your spot.

**Claude + Browser Automation: Your AI Sees Your Screen** — Claude can now see your browser. It reads pages, clicks buttons, fills forms, takes screenshots, and navigates sites — all from a conversation or a script. This isn't a future feature. It's live right now.

Three tools make this work: **Claude Computer Use** (full desktop control — screenshot, mouse, keyboard), **Claude in Chrome MCP** (browser-native tools — navigate, read pages, click, type), and **Puppeteer MCP** (headless automation for scripts and pipelines). Each solves different problems. This event shows you when to use which.

This is a live demo workshop. We'll automate real browser tasks from scratch — starting with simple page reads and ending with multi-step workflows that run unattended.

**What we'll cover:**

**🖥️ The Browser Automation Stack Inside Claude**

How Computer Use, Chrome MCP, and Puppeteer MCP work together — and when to use browser automation vs direct MCP integrations vs APIs.

**🔧 Demos We'll Build Live:**

* **Page Reader & Data Extractor** — point Claude at any webpage, extract structured data into a spreadsheet. No scraping code required.
* **Form Filler & Submission Bot** — automate repetitive form submissions across multiple sites using Claude in Chrome
* **Research Automator** — give Claude a research question, watch it open tabs, read pages, take notes, and compile a summary
* **Visual Testing Agent** — screenshot pages, compare layouts, flag visual regressions automatically
* **Website Poster** — post content to platforms that don't have API connectors using Puppeteer browser automation
* **Multi-Step Workflow** — chain browser actions: log into a site → navigate to dashboard → extract report → save to Google Drive → send summary via email

**🔒 Security & Permission Models**

What Claude can and can't see in your browser. Permission scoping, supervision best practices, and why browser agents need guardrails. Computer Use is still a research preview — we'll be honest about the limitations.

**⚠️ Key Caveat:** Claude Computer Use requires macOS. Chrome MCP and Puppeteer MCP work on all platforms. We'll cover all three so everyone leaves with something they can use immediately.

**Who this is for:** Anyone who spends time on repetitive browser tasks — filling forms, extracting data, posting content, running reports. Business owners, marketers, researchers, and developers alike. If you copy-paste between browser tabs, this event automates that. Prior experience with Claude is helpful but not required.

**Who this is NOT for:** People looking for a general AI overview. This is a hands-on workshop. Every demo involves real browser automation. If you want theory, see our earlier events in the series.

📚 **Free resources at the event:**

* Setup guide — follow along with every demo live
* Homework doc — extended exercises to keep building after the event
* Slide deck — full reference for everything we cover
